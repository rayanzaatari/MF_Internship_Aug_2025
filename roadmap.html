Software Development Internship Program

An 8-week immersive experience designed to build production-ready full-stack developers. This section provides a high-level summary of the program's goals and the exciting project interns will develop.

Program Overview

This internship provides hands-on, industry-grade experience in developing a complete web application using ASP.NET Core 9.0 and SQL Server. Participants work collaboratively, emphasizing clean architecture, RESTful APIs, and version control with GitHub.

Project Theme: "SkillBridge"

Interns will build "SkillBridge," an Internship & Training Management System. This real-world project serves as the primary vehicle for learning and applying new skills throughout the program.

Key Features:

  • Organizations can post opportunities.
  • Students can discover and apply for internships.
  • Coordinators can track attendance and evaluations.
  • Role-based access for Admins, Coordinators, and Students.

Intern Roles & Collaboration

To simulate a real-world development environment, interns will be assigned to specific roles within the project team. This fosters collaboration, specialization, and a deeper understanding of the software development lifecycle.

Team Lead

Coordinates task delegation, tracks progress, and manages GitHub merging.

Backend Developer

Handles API development, database design, and application logic.

Frontend Developer

Builds the user interface with Razor, handles form logic, and visuals.

Mobile Developer

(Optional) Works on the React Native mobile application if time allows.
